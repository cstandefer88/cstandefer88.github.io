Xpndbl

Code on GitHub

Link to App

Xpndbl ("expendable" without the vowels) is an end-of-month personal finance tracker allowing users to enter in the month in question, income for the month, expenses, and then save them to a database. Users can then review past months if they desire to compare and contrast previous months with more recent ones. This was a solo project and my capstone for General Assembly's Web Development Immersive in our final project week. It was built in 4 days using the MERN Stack (MongoDB, Express, React, and Node) - as opposed to the more well-known MEAN Stack. This is the MVP. I still have many features to build out on this for it to be complete as well as revamping the design.
